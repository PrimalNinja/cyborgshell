Datei-Befehle  
Diese Befehle beziehen sich auf Dateien in Ihrem aktuellen Dateispeicher.

beautify <filename>

	Verschönert eine JSON-Datei, ohne das aktuell geladene Programm zu beschädigen.
	
dir [<filename>] (oder cat, ls)

	Auflistung der gespeicherten Programme. Der Dateiname kann Wildcards * oder ? enthalten.
	
copy <filenamea> <filenameb> (oder cp)

	Kopiert eine Datei von filenamea nach filenameb.
	
del <filename> (oder delete, era)

	Löscht ein Programm.
	
load <filename>

	Lädt ein Programm.
	
ren <filenamea> <filenameb> (oder rename, mv)

	Ändert den Namen einer Datei von filenamea nach filenameb.
	
run <filename>

	Führt ein Programm aus und ersetzt das aktuell geladene Programm.
	
save <filename>

	Speichert ein Programm.
	
space [<spacekey> | <alias>] (oder cs, cd, chdir)

	Zeigt Ihren aktuellen Speicherort an oder wechselt zu einem neuen.
	
spaces

	Auflistung aller Dateispeicherorte, auf die Sie Zugriff haben.

type <filename> [[fromline]-[toline]]

	Gibt ein Programm aus, ohne das aktuell geladene Programm zu beschädigen.
