Αναφορά API

Προς το παρόν είναι μόνο τα εξής:

(σχετικά με την αποσφαλμάτωση)

api.debugObject(<object>, [<recurseflag>]);

	εμφανίζει το παρεχόμενο αντικείμενο, αν η αναδρομική σημαία παραληφθεί, προεπιλέγεται σε true.

(σχετικά με την οθόνη)

api.cls();

	καθαρίζει την οθόνη

api.print(<output>);

	εκτυπώνει κάποια έξοδο
