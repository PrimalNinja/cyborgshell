Εντολές Επεξεργαστή  
Αυτές οι εντολές σας βοηθούν να επεξεργαστείτε το τρέχον πρόγραμμα σας.

beautify

	Ομορφαίνει το φορτωμένο JSON.

cls (ή clear)

	Καθαρίζει την οθόνη.

edit <linenumber> (ή ed)

	Επεξεργάζεται μια γραμμή με τον επεξεργαστή γραμμών.

file [<filenumber>]

	Εμφανίζει το τρέχον αρχείο για το οποίο ισχύουν οι εντολές επεξεργαστή ή ορίζει ένα νέο.

filename [<filename>]

	Εμφανίζει το όνομα αρχείου του τρέχοντος αρχείου ή το ορίζει. Αυτό το όνομα αρχείου είναι για τον επεξεργαστή, δεν δίνεται στα αρχεία που φορτώνετε ή αποθηκεύετε, αλλά θα οριστεί από εσάς όταν φορτώνετε ή αποθηκεύετε αρχεία.

files

	Λίστα όλων των αρχείων στον επεξεργαστή με τους σχετικούς αριθμούς αρχείων.

list [[fromline]-[toline]]

	Λίστα του προγράμματος. Παραδείγματα: list 30-, list -30, list 20-30, list, list 40.

new

	Καθαρίζει το τρέχον πρόγραμμα από τον επεξεργαστή κώδικα.

newfile

	Δημιουργεί ένα νέο αρχείο μέσα στον επεξεργαστή σας.

paste <linenumber> <filenumber>

	Επικολλά το περιεχόμενο του καθορισμένου αριθμού αρχείου στο τρέχον αρχείο πριν από τον καθορισμένο αριθμό γραμμής. Αυτή η δυνατότητα επικόλλησης σας επιτρέπει να χρησιμοποιείτε πολλαπλά αποσπάσματα κώδικα που έχουν φορτωθεί στον επεξεργαστή ως ξεχωριστές πρόχειρες και να τα επικολλάτε όπως απαιτείται στον κώδικά σας.

renum

	Αριθμεί ξανά το πρόγραμμα. Σημείωση: Η JavaScript δεν χρησιμοποιεί αριθμούς γραμμών, αλλά τους χρησιμοποιούμε για σκοπούς επεξεργασίας.

run

	Εκτελεί το τρέχον πρόγραμμα.
