Εντολές Αρχείων  
Αυτές οι εντολές σχετίζονται με αρχεία στον τρέχοντα χώρο αρχείων σας.

beautify <filename>

	Ομορφαίνει ένα αρχείο JSON χωρίς να διαφθείρει το πρόγραμμα που είναι φορτωμένο αυτή τη στιγμή.
	
dir [<filename>] (ή cat, ls)

	Λίστα με αποθηκευμένα προγράμματα. Το όνομα αρχείου μπορεί να περιέχει χαρακτήρες μπαλαντέρ * ή ?.
	
copy <filenamea> <filenameb> (ή cp)

	Αντιγράφει ένα αρχείο από το filenamea στο filenameb.
	
del <filename> (ή delete, era)

	Διαγράφει ένα πρόγραμμα.
	
load <filename>

	Φορτώνει ένα πρόγραμμα.
	
ren <filenamea> <filenameb> (ή rename, mv)

	Μετονομάζει ένα αρχείο από το filenamea στο filenameb.
	
run <filename>

	Εκτελεί ένα πρόγραμμα, αντικαθιστώντας το υπάρχον φορτωμένο πρόγραμμα.
	
save <filename>

	Αποθηκεύει ένα πρόγραμμα.
	
space [<spacekey> | <alias>] (ή cs, cd, chdir)

	Εμφανίζει το τρέχον χώρο σας ή αλλάζει σε έναν νέο.
	
spaces

	Λίστα με όλους τους χώρους αρχείων στους οποίους έχετε πρόσβαση.

type <filename> [[fromline]-[toline]]

	Πληκτρολογεί ένα πρόγραμμα χωρίς να διαφθείρει το πρόγραμμα που είναι φορτωμένο αυτή τη στιγμή.
