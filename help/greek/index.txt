Καλώς ήρθατε στο Home Computer JS!

Αυτός ο οδηγός βοήθειας θα σας καθοδηγήσει μέσω των διαφόρων εντολών και χαρακτηριστικών που είναι διαθέσιμα.

Βεβαιωθείτε ότι έχετε διαβάσει την ενότητα σχετικά με εμάς πληκτρολογώντας 'help about' και διαβάστε την πολιτική απορρήτου πληκτρολογώντας 'help privacy' καθώς και τους όρους και τις προϋποθέσεις πληκτρολογώντας 'help terms'.
(βασικά, αν χρησιμοποιείτε αυτό το προϊόν, συμφωνείτε με την πολιτική απορρήτου καθώς και με τους όρους και τις προϋποθέσεις)

Μπορείτε να εγγραφείτε ώστε να μπορείτε να συνδεθείτε και να προγραμματίσετε τα δικά σας προγράμματα και να τα μοιραστείτε με άλλους.
Πληκτρολογήστε 'help account' για βοήθεια σχετικά με τις εντολές εγγραφής και σύνδεσης καθώς και με κάποιες άλλες εντολές λογαριασμού.

Βοήθεια Ευρετήριο

help

    Εμφανίζει αυτό το ευρετήριο βοήθειας.

help general

    Εμφανίζει βοήθεια για γενικές εντολές.

help editor

    Εμφανίζει βοήθεια για τις εντολές του επεξεργαστή.

help file

    Εμφανίζει βοήθεια για τις εντολές αρχείου.

help account

    Εμφανίζει βοήθεια για τις εντολές λογαριασμού.

help sharing

    Εμφανίζει βοήθεια για τις εντολές κοινής χρήσης.

help about

    Εμφανίζει κάποιες πληροφορίες σχετικά με αυτό το προϊόν.

help roadmap

    Εμφανίζει κάποιες πληροφορίες σχετικά με το τι σχεδιάζουμε να κάνουμε με αυτό το προϊόν.

help privacy

    Εμφανίζει την πολιτική απορρήτου.

help terms

    Εμφανίζει τους όρους και τις προϋποθέσεις.

help etiquette

    Εμφανίζει τους κανόνες ευγένειας για αυτό το δημόσιο σύστημα.
