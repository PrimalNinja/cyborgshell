API संदर्भ

अभी के लिए यह सिर्फ निम्नलिखित है:

(debug संबंधित)

api.debugObject(<object>, [<recurseflag>]);

	प्रदान किए गए ऑब्जेक्ट को प्रदर्शित करें, यदि पुनरावृत्ति ध्वज छोड़ा गया है, तो इसका डिफ़ॉल्ट मान सत्य है।

(screen संबंधित)

api.cls();

	स्क्रीन साफ़ करें

api.print(<output>);

	कुछ आउटपुट प्रिंट करें
