エディターコマンド  
これらのコマンドは、現在のプログラムを編集するのに役立ちます。

beautify

	読み込まれたJSONを整形します。

cls (または clear)

	画面をクリアします。

edit <linenumber> (または ed)

	行エディタで行を編集します。

file [<filenumber>]

	エディターコマンドが適用される現在のファイルを表示するか、新しいファイルを設定します。

filename [<filename>]

	現在のファイルのファイル名を表示するか、設定します。このファイル名はエディター用であり、読み込んだり保存したりするファイルには与えられませんが、ファイルを読み込んだり保存したりすることで設定されます。

files

	エディター内のすべてのファイルを関連するファイル番号と共にリストします。

list [[fromline]-[toline]]

	プログラムをリストします。例: list 30-, list -30, list 20-30, list, list 40。

new

	コードエディタから現在のプログラムをクリアします。

newfile

	エディター内に新しいファイルを作成します。

paste <linenumber> <filenumber>

	指定されたファイル番号の内容を、指定された行番号の前に現在のファイルに貼り付けます。この貼り付け機能により、エディターに読み込まれた複数のコードフラグメントを別々のクリップボードとして使用し、必要に応じてコードに貼り付けることができます。

renum

	プログラムの行番号を再割り当てします。注: JavaScriptは行番号を使用しませんが、編集の目的で使用します。

run

	現在のプログラムを実行します。
