편집기 명령어  
이 명령어들은 현재 프로그램을 편집하는 데 도움을 줍니다.

beautify

	로딩된 JSON을 아름답게 만듭니다.

cls (또는 clear)

	화면을 지웁니다.

edit <linenumber> (또는 ed)

	라인 편집기를 사용하여 한 줄을 편집합니다.

file [<filenumber>]

	편집기 명령어가 적용되는 현재 파일을 보여주거나 새 파일로 설정합니다.

filename [<filename>]

	현재 파일의 파일 이름을 보여주거나 설정합니다. 이 파일 이름은 편집기를 위한 것이며, 로드하거나 저장하는 파일에 제공되지 않고, 대신 파일을 로드하거나 저장함으로써 설정됩니다.

files

	편집기에서 관련된 파일 번호와 함께 모든 파일을 나열합니다.

list [[fromline]-[toline]]

	프로그램을 나열합니다. 예: list 30-, list -30, list 20-30, list, list 40.

new

	코드 편집기에서 현재 프로그램을 지웁니다.

newfile

	편집기 내에서 새 파일을 생성합니다.

paste <linenumber> <filenumber>

	지정된 파일 번호의 내용을 현재 파일의 지정된 줄 번호 앞에 붙여넣습니다. 이 붙여넣기 기능은 편집기에 로드된 여러 코드 조각을 별도의 클립보드로 사용하고 필요에 따라 코드에 붙여넣을 수 있게 해줍니다.

renum

	프로그램의 번호를 다시 매깁니다. 참고: JavaScript는 줄 번호를 사용하지 않지만, 우리는 편집 목적으로 사용합니다.

run

	현재 프로그램을 실행합니다.
