API Справочник

На данный момент это всего лишь следующее:

(связано с отладкой)

api.debugObject(<object>, [<recurseflag>]);

	отобразить предоставленный объект, если флаг рекурсии опущен, по умолчанию он равен true.

(связано с экраном)

api.cls();

	очистить экран

api.print(<output>);

	вывести некоторый результат
