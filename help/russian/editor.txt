Команды редактора  
Эти команды помогают вам редактировать вашу текущую программу.

beautify

	У Beautifies загруженный JSON.

cls (или clear)

	Очищает экран.

edit (или ed)

	Редактирует строку с помощью редактора строк.

file [<filenumber>]

	Показывает ваш текущий файл, для которого применяются команды редактора, или устанавливает новый.

filename [<filename>]

	Показывает имя файла текущего файла или устанавливает его. Это имя файла для редактора, оно не передается файлам, которые вы загружаете или сохраняете, а скорее будет установлено вами при загрузке или сохранении файлов.

files

	Список всех файлов в редакторе с их соответствующими номерами файлов.

list [[fromline]-[toline]]

	Перечисляет программу. Примеры: list 30-, list -30, list 20-30, list, list 40.

new

	Очищает текущую программу из редактора кода.

newfile

	Создает новый файл в вашем редакторе.

paste <linenumber> <filenumber>

	Вставляет содержимое указанного номера файла в текущий файл перед указанным номером строки. Эта функция вставки позволяет вам использовать несколько фрагментов кода, загруженных в редактор, в качестве отдельных буферов обмена и вставлять их по мере необходимости в ваш код.

renum

	Перенумеровывает программу. Примечание: JavaScript не использует номера строк, но мы используем их для целей редактирования.

run

	Запускает текущую программу.
