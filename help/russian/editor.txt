Команды редактора  
Эти команды помогают вам редактировать вашу текущую программу.

beautify

	У Beautifies загруженный JSON.

cls (или clear)

	Очищает экран.

edit (или ed)

	Редактирует строку с помощью редактора строк.

list [[fromline]-[toline]]

	Перечисляет программу. Примеры: list 30-, list -30, list 20-30, list, list 40.

new

	Очищает текущую программу из редактора кода.

renum

	Перенумеровывает программу. Примечание: JavaScript не использует номера строк, но мы используем их для целей редактирования.

run

	Запускает текущую программу.
