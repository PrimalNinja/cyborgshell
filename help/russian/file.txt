Команды файлов  
Эти команды относятся к файлам в вашем текущем файловом пространстве.  

beautify <filename>  

	Красит файл JSON, не повреждая текущую загруженную программу.  

dir [<filename>] (или cat, ls)  

	Список сохранённых программ. Имя файла может содержать подстановочные знаки * или ?.  

copy <filenamea> <filenameb> (или cp)  

	Копирует файл из filenamea в filenameb.  

del <filename> (или delete, era)  

	Удаляет программу.  

load <filename>  

	Загружает программу.  

ren <filenamea> <filenameb> (или rename, mv)  

	Переименовывает файл из filenamea в filenameb.  

run <filename>  

	Запускает программу, заменяя существующую загруженную программу.  

save <filename>  

	Сохраняет программу.  

space [<spacekey> | <alias>] (или cs, cd, chdir)  

	Показывает ваше текущее пространство или меняет на новое.  

spaces  

	Список всех файловых пространств, к которым у вас есть доступ.  

type <filename> [[fromline]-[toline]]  

	Выводит программу, не повреждая текущую загруженную программу.
