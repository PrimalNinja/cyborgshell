Добро пожаловать в Home Computer JS!

Этот справочник поможет вам ознакомиться с различными командами и функциями, доступными в системе.

Не забудьте прочитать информацию о продукте, введя 'help about', а также ознакомиться с политикой конфиденциальности, введя 'help privacy', и с условиями использования, введя 'help terms'.
(по сути, если вы используете этот продукт, вы соглашаетесь с политикой конфиденциальности и условиями использования)

Вы можете зарегистрироваться, чтобы войти в систему и программировать свои собственные программы, а также делиться ими с другими.
Введите 'help account', чтобы получить помощь по командам регистрации и входа, а также другим командам, связанным с учетной записью.

Справочный индекс

help

	Показывает этот справочный индекс.

help general

	Показывает помощь по общим командам.
	
help editor

	Показывает помощь по командам редактора.
	
help file

	Показывает помощь по командам работы с файлами.
	
help account

	Показывает помощь по командам учетной записи.
	
help sharing

	Показывает помощь по командам совместного использования.

help about

	Показывает информацию об этом продукте.
	
help roadmap

	Показывает информацию о том, что мы планируем сделать с этим продуктом.
	
help privacy

	Показывает политику конфиденциальности.

help terms

	Показывает условия использования.

help etiquette

	Показывает правила этикета для этой публичной системы.
