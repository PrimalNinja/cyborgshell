เอกสารอ้างอิง API

ตอนนี้มีเพียงสิ่งต่อไปนี้:

(เกี่ยวกับการดีบัก)

api.debugObject(<object>, [<recurseflag>]);

	แสดงวัตถุที่ให้มา หากไม่มีการระบุธงการทำซ้ำ มันจะตั้งค่าเป็น true โดยอัตโนมัติ

(เกี่ยวกับหน้าจอ)

api.cls();

	เคลียร์หน้าจอ

api.print(<output>);

	พิมพ์ข้อมูลบางอย่าง
