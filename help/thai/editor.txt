คำสั่งของโปรแกรมแก้ไข  
คำสั่งเหล่านี้ช่วยให้คุณแก้ไขโปรแกรมปัจจุบันของคุณ  

beautify  

	ปรับรูปแบบ JSON ที่โหลดมาให้สวยงาม  

cls (หรือ clear)  

	เคลียร์หน้าจอ  

edit <linenumber> (หรือ ed)  

	แก้ไขบรรทัดด้วยโปรแกรมแก้ไขบรรทัด  

file [<filenumber>]  

	แสดงไฟล์ปัจจุบันของคุณซึ่งคำสั่งแก้ไขใช้ได้หรือกำหนดไฟล์ใหม่  

filename [<filename>]  

	แสดงชื่อไฟล์ของไฟล์ปัจจุบันหรือกำหนดชื่อใหม่ ชื่อไฟล์นี้ใช้สำหรับโปรแกรมแก้ไข ไม่ได้ใช้กับไฟล์ที่คุณโหลดหรือบันทึก แต่จะถูกตั้งโดยคุณเมื่อโหลดหรือบันทึกไฟล์  

files  

	แสดงรายการไฟล์ทั้งหมดในโปรแกรมแก้ไขพร้อมหมายเลขไฟล์ที่เกี่ยวข้อง  

list [[fromline]-[toline]]  

	แสดงรายการโปรแกรม ตัวอย่าง: list 30-, list -30, list 20-30, list, list 40  

new  

	เคลียร์โปรแกรมปัจจุบันจากโปรแกรมแก้ไขโค้ด  

newfile  

	สร้างไฟล์ใหม่ภายในโปรแกรมแก้ไขของคุณ  

paste <linenumber> <filenumber>  

	วางเนื้อหาของหมายเลขไฟล์ที่ระบุลงในไฟล์ปัจจุบันก่อนหมายเลขบรรทัดที่ระบุ ความสามารถในการวางนี้ช่วยให้คุณสามารถใช้หลายส่วนของโค้ดที่โหลดเข้ามาในโปรแกรมแก้ไขเป็นคลิปบอร์ดแยกต่างหากและวางตามที่ต้องการในโค้ดของคุณ  

renum  

	กำหนดหมายเลขใหม่ให้กับโปรแกรม หมายเหตุ: JavaScript ไม่ใช้หมายเลขบรรทัด แต่เราใช้เพื่อวัตถุประสงค์ในการแก้ไข  

run  

	รันโปรแกรมปัจจุบัน
