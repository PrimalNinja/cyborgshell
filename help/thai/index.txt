ยินดีต้อนรับสู่ Home Computer JS!

คู่มือนี้จะช่วยแนะนำคุณเกี่ยวกับคำสั่งและฟีเจอร์ต่าง ๆ ที่มีอยู่

โปรดอ่านข้อมูลเกี่ยวกับโดยพิมพ์ 'help about' และอ่านนโยบายความเป็นส่วนตัวโดยพิมพ์ 'help privacy' รวมถึงข้อกำหนดและเงื่อนไขโดยพิมพ์ 'help terms' 
(โดยพื้นฐานแล้วถ้าคุณใช้ผลิตภัณฑ์นี้ คุณยอมรับนโยบายความเป็นส่วนตัวและข้อกำหนดและเงื่อนไข)

คุณสามารถลงทะเบียนเพื่อให้คุณสามารถเข้าสู่ระบบและเขียนโปรแกรมของคุณเองและแชร์กับผู้อื่น
พิมพ์ 'help account' เพื่อขอความช่วยเหลือเกี่ยวกับคำสั่งลงทะเบียนและเข้าสู่ระบบรวมถึงคำสั่งบัญชีอื่น ๆ

ดัชนีช่วยเหลือ

help

	แสดงดัชนีช่วยเหลือนี้

help general

	แสดงความช่วยเหลือเกี่ยวกับคำสั่งทั่วไป
	
help editor

	แสดงความช่วยเหลือเกี่ยวกับคำสั่งแก้ไข
	
help file

	แสดงความช่วยเหลือเกี่ยวกับคำสั่งไฟล์
	
help account

	แสดงความช่วยเหลือเกี่ยวกับคำสั่งบัญชี
	
help sharing

	แสดงความช่วยเหลือเกี่ยวกับคำสั่งการแชร์

help about

	แสดงข้อมูลบางอย่างเกี่ยวกับผลิตภัณฑ์นี้
	
help roadmap

	แสดงข้อมูลบางอย่างเกี่ยวกับสิ่งที่เราวางแผนจะทำกับผลิตภัณฑ์นี้
	
help privacy

	แสดงนโยบายความเป็นส่วนตัว

help terms

	แสดงข้อกำหนดและเงื่อนไข

help etiquette

	แสดงกฎระเบียบมารยาทสำหรับระบบสาธารณะนี้
